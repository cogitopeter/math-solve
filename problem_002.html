# 002
2026-05-27 来源：作业 中档

一、解题思路

审题 — 识别题型
看到 $4S_n = (a_n + 1)^2$，这是典型的"$S_n$ 与 $a_n$ 关系"题型。条件里没有直接给出 $a_n$ 的递推式，但 $S_n$ 和 $a_n$ 绑在一起。第一反应：用 $a_n = S_n - S_{n-1}$（$n \geqslant 2$）把 $S_n$ 消掉，把问题转化为纯粹的 $a_n$ 递推关系。
推进 — 相邻两式作差
写出 $n$ 和 $n-1$ 的等式：$4S_n = (a_n+1)^2$，$4S_{n-1} = (a_{n-1}+1)^2$。相减得 $4a_n = (a_n+1)^2 - (a_{n-1}+1)^2$。移项整理后发现 $(a_n - 1)^2 = (a_{n-1}+1)^2$。到这里要开方了——但平方等于平方，开方之后有正负两种情况，不能漏掉。
关键洞察 — 用"各项为正"排除一个分支
开方得 $a_n - 1 = \pm(a_{n-1} + 1)$。取负号则 $a_n = -a_{n-1}$，但题目说"各项均为正数"，相邻两项不可能异号，所以负号分支不可能。只剩 $a_n - 1 = a_{n-1} + 1$，即 $a_n = a_{n-1} + 2$。这就是公差为 2 的等差数列！但别忘了：$n=1$ 要单独验证。
第(2)问 — 分组求和的直觉
得到 $a_n = 2n-1$ 后，$b_n = (2n-1) + 2^{2n-1}$。一个多项式、一个指数，两类完全不同的项。自然想到分组求和：多项式部分 $\sum(2k-1) = n^2$（前 $n$ 个奇数之和是完全平方），指数部分 $2^1 + 2^3 + \cdots + 2^{2n-1}$ 是公比为 $4$ 的等比数列。

二、解题过程

解法一：作差递推法 通法
第 (1) 问：求通项公式
1. 当 $n \geqslant 2$ 时，由 $4S_n = (a_n + 1)^2$ 和 $4S_{n-1} = (a_{n-1} + 1)^2$ 相减： $$4a_n = (a_n + 1)^2 - (a_{n-1} + 1)^2$$
2. 展开左边：$(a_n + 1)^2 = a_n^2 + 2a_n + 1$，代入： $$4a_n = a_n^2 + 2a_n + 1 - (a_{n-1} + 1)^2$$ 移项得 $$a_n^2 - 2a_n + 1 = (a_{n-1} + 1)^2 \implies (a_n - 1)^2 = (a_{n-1} + 1)^2$$
3. 开平方：$a_n - 1 = \pm(a_{n-1} + 1)$。
若取负号：$a_n = -a_{n-1}$，但各项为正，矛盾。
故 $a_n - 1 = a_{n-1} + 1$，即 $a_n = a_{n-1} + 2$（$n \geqslant 2$）。
4. 当 $n = 1$：$4a_1 = (a_1 + 1)^2$，展开得 $a_1^2 - 2a_1 + 1 = 0$，即 $(a_1 - 1)^2 = 0$，解得 $a_1 = 1$。
5. 因此 $\{a_n\}$ 是首项为 $1$、公差为 $2$ 的等差数列： $$\boxed{a_n = 2n - 1}$$
第 (2) 问：求 $T_n$
6. $b_n = a_n + 2^{a_n} = (2n - 1) + 2^{2n-1}$，分组求和： $$T_n = \sum_{k=1}^{n} (2k-1) + \sum_{k=1}^{n} 2^{2k-1}$$
7. 第一部分：$\displaystyle\sum_{k=1}^{n} (2k-1) = 1 + 3 + 5 + \cdots + (2n-1) = n^2$。
8. 第二部分：$2^1 + 2^3 + 2^5 + \cdots + 2^{2n-1}$ 是首项 $a_1 = 2$、公比 $q = \dfrac{2^3}{2^1} = 4$、共 $n$ 项的等比数列： $$\sum_{k=1}^{n} 2^{2k-1} = \frac{2(4^n - 1)}{4 - 1} = \frac{2(4^n - 1)}{3}$$
9. 因此 $$\boxed{T_n = n^2 + \frac{2(4^n - 1)}{3}}$$
解法二：归纳验证法 辅助验证
1. 逐项计算：$n=1$：$4a_1 = (a_1+1)^2 \Rightarrow a_1 = 1$。
2. $n=2$：$4(1+a_2) = (a_2+1)^2$，展开得 $a_2^2 - 2a_2 - 3 = 0$，即 $(a_2-3)(a_2+1) = 0$，取正值 $a_2 = 3$。
3. $n=3$：$4(1+3+a_3) = (a_3+1)^2$，展开得 $a_3^2 - 2a_3 - 15 = 0$，即 $(a_3-5)(a_3+3) = 0$，取正值 $a_3 = 5$。
4. 猜想 $a_n = 2n-1$，验证：$S_n = \sum(2k-1) = n^2$，$4S_n = 4n^2$，$(a_n+1)^2 = (2n)^2 = 4n^2$。等式成立。
这种方法适合考试中快速验证答案是否正确。若需严格证明，可用数学归纳法补充。第 (2) 问的求和过程与解法一相同。

四、思维方式

$S_n$ 与 $a_n$ 混合出现时，作差是消去 $S_n$ 的标准操作

平方相等不等于底数相等，必须考虑正负两种情况

题目的隐含条件是筛选解的关键，不能忽视

【$n=1$ 时 $a_n = S_n - S_{n-1}$ 不适用，需单独代入原式求 $a_1$】
$S_n$ 与 $a_n$ 关系中，$n=1$ 永远要单独验证是否一致
分类讨论
【$b_n = (2n-1) + 2^{2n-1}$，拆成多项式和指数两组分别求和】

指数的底数相同但指数成等差，等价于公比为 $2^d$ 的等比数列

五、易错点与陷阱

忘记验证 $n=1$ 的情况，直接认为递推式 $a_n = a_{n-1} + 2$ 对所有 $n$ 成立
$a_n = S_n - S_{n-1}$ 只对 $n \geqslant 2$ 成立，$n=1$ 必须代入原式 $4a_1 = (a_1+1)^2$ 单独求解
如果 $n=1$ 算出的 $a_1$ 与递推式给出的不一致，通项公式需要分段表达
开方时漏掉负号分支，直接写 $a_n - 1 = a_{n-1} + 1$
$(a_n-1)^2 = (a_{n-1}+1)^2$ 应先写出 $a_n - 1 = \pm(a_{n-1}+1)$，再用"各项为正"排除负号
考试中不写讨论过程会被扣分。即使结果正确，推理过程不完整也会丢 1-2 分
等比数列公比算错：认为 $2^{2k-1}$ 的公比是 $2$
公比 $q = \dfrac{2^{2(k+1)-1}}{2^{2k-1}} = \dfrac{2^{2k+1}}{2^{2k-1}} = 2^2 = 4$
指数差是 $2$，所以公比是 $2^2 = 4$，不是底数 $2$。可以用相邻两项 $2^3 / 2^1 = 4$ 快速验证
$\sum(2k-1)$ 用错公式，写成 $\dfrac{n(2n-1)}{2}$
$\displaystyle\sum_{k=1}^{n}(2k-1) = 2 \cdot \frac{n(n+1)}{2} - n = n^2$，或直接记住"前 $n$ 个奇数之和等于 $n^2$"
这是一个高频结论，建议直接记忆：$1+3+5+\cdots+(2n-1) = n^2$

六、变式题方向

变式一：改变 $S_n$ 与 $a_n$ 的关系
若改为 $S_n = \dfrac{1}{4}(a_n^2 + 4n - 1)$，或 $2S_n = a_n \cdot a_{n+1}$，做法类似但递推式不同。
核心区别：不同的 $S_n$-$a_n$ 关系可能导致等差、等比或更复杂的数列，需要灵活应对。
变式二：$b_n$ 改为需要错位相减的形式
若 $b_n = a_n \cdot 2^{a_n} = (2n-1) \cdot 2^{2n-1}$（乘积而非加和），则求和不能分组，需要用错位相减法。
核心区别：多项式与指数的"加"可以分组，"乘"必须用错位相减。这是两种求和方法的典型分界。
变式三：去掉"各项为正"条件
若不限制各项为正，$(a_n-1)^2 = (a_{n-1}+1)^2$ 的两个分支都可能成立，通项公式可能是分段函数或交替数列。
核心区别：条件的强弱直接影响解的唯一性，训练审题时对隐含条件的敏感度。
