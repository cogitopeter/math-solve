# 004
2026-05-27 来源：作业 中档
已知数列 $\{a_n\}$ 的前 $n$ 项和 $S_n = -\dfrac{1}{2}n^2 + kn$（其中 $k \in \mathbf{N}_+$），且 $S_n$ 的最大值为 $8$。
（I）确定常数 $k$，并求 $a_n$；
（II）求数列 $\left\{\dfrac{9-2a_n}{2^n}\right\}$ 的前 $n$ 项和 $T_n$。

一、解题思路

审题 — 抓住两个关键信息
$S_n = -\frac{1}{2}n^2 + kn$ 是关于 $n$ 的二次函数，开口朝下（$-\frac{1}{2} < 0$）。 题目说"$S_n$ 的最大值为 $8$"——二次函数求最值，自然想到配方或者顶点公式。 又因为 $k \in \mathbf{N}_+$，$k$ 是正整数，这会简化取最值的讨论。
联想 — $S_n$ 与 $a_n$ 的关系
第 (I) 问求 $a_n$，标准套路：$a_n = S_n - S_{n-1}$（$n \geq 2$），再单独验证 $n=1$。 这里 $S_n$ 是二次式，差分后 $a_n$ 应该是一次式（等差数列的特征）。
突破 — 第 (II) 问的化简是关键
看到 $\frac{9-2a_n}{2^n}$，先别急着套公式。把 $a_n$ 代入化简，$9 - 2a_n = 9 - (9-2n) = 2n$， 于是通项变成 $\frac{2n}{2^n} = \frac{n}{2^{n-1}}$。这是"$n$ 乘以等比"的形式，错位相减法直接上。
路径 — 两问的方法都很明确
第 (I) 问：配方求 $k$ → 差分求 $a_n$。第 (II) 问：代入化简 → 错位相减法。 没有岔路，计算准确即可。

二、解题过程

解法一 通法 · 配方 + 错位相减
第 (I) 问：求 $k$ 和 $a_n$
Step 1：对 $S_n$ 配方求最值
$$S_n = -\frac{1}{2}n^2 + kn = -\frac{1}{2}(n^2 - 2kn) = -\frac{1}{2}(n - k)^2 + \frac{k^2}{2}$$
由于 $-\frac{1}{2} < 0$，$S_n$ 作为 $n$ 的二次函数开口朝下，对称轴为 $n = k$。
因为 $k \in \mathbf{N}_+$，当 $n = k$ 时，$S_n$ 取得最大值： $$S_k = \frac{k^2}{2} = 8 \implies k^2 = 16 \implies k = 4$$ （$k \in \mathbf{N}_+$，故舍去 $k = -4$）
Step 2：求 $a_n$
此时 $S_n = -\frac{1}{2}n^2 + 4n$。
当 $n \geq 2$ 时： $$a_n = S_n - S_{n-1} = \left(-\frac{n^2}{2} + 4n\right) - \left(-\frac{(n-1)^2}{2} + 4(n-1)\right)$$ $$= -\frac{n^2}{2} + 4n + \frac{(n-1)^2}{2} - 4(n-1)$$ $$= \frac{-(n^2) + (n-1)^2}{2} + 4 = \frac{-(2n-1)}{2} + 4 = \frac{9 - 2n}{2}$$
当 $n = 1$ 时：$a_1 = S_1 = -\frac{1}{2} + 4 = \frac{7}{2}$，代入公式 $\frac{9-2}{2} = \frac{7}{2}$，一致。
$$\boxed{k = 4, \quad a_n = \frac{9 - 2n}{2}}$$
第 (II) 问：求 $T_n$
Step 3：化简通项
$$\frac{9 - 2a_n}{2^n} = \frac{9 - 2 \cdot \frac{9-2n}{2}}{2^n} = \frac{9 - (9 - 2n)}{2^n} = \frac{2n}{2^n} = \frac{n}{2^{n-1}}$$
Step 4：错位相减法
设 $b_n = \frac{n}{2^{n-1}}$，则： $$T_n = \frac{1}{1} + \frac{2}{2} + \frac{3}{4} + \frac{4}{8} + \cdots + \frac{n}{2^{n-1}}$$
两边乘以 $\frac{1}{2}$： $$\frac{1}{2}T_n = \frac{1}{2} + \frac{2}{4} + \frac{3}{8} + \cdots + \frac{n-1}{2^{n-1}} + \frac{n}{2^n}$$
作差（$T_n - \frac{1}{2}T_n$）： $$\frac{1}{2}T_n = \underbrace{1 + \frac{1}{2} + \frac{1}{4} + \cdots + \frac{1}{2^{n-1}}}_{\text{等比数列求和}} - \frac{n}{2^n}$$
等比数列部分：首项 $1$，公比 $\frac{1}{2}$，共 $n$ 项： $$1 + \frac{1}{2} + \cdots + \frac{1}{2^{n-1}} = \frac{1 - \left(\frac{1}{2}\right)^n}{1 - \frac{1}{2}} = 2\left(1 - \frac{1}{2^n}\right) = 2 - \frac{1}{2^{n-1}}$$
代入： $$\frac{1}{2}T_n = 2 - \frac{1}{2^{n-1}} - \frac{n}{2^n} = 2 - \frac{2}{2^n} - \frac{n}{2^n} = 2 - \frac{n+2}{2^n}$$
$$\boxed{T_n = 4 - \frac{n+2}{2^{n-1}}}$$
验证：
$n=1$：$T_1 = \frac{1}{1} = 1$，公式 $= 4 - \frac{3}{1} = 1$ ✓
$n=2$：$T_2 = 1 + 1 = 2$，公式 $= 4 - \frac{4}{2} = 2$ ✓
$n=3$：$T_3 = 1 + 1 + \frac{3}{4} = \frac{11}{4}$，公式 $= 4 - \frac{5}{4} = \frac{11}{4}$ ✓
解法二 $a_n$ 正负性判定最值
第 (I) 问另解：用 $a_n$ 的正负性确定 $S_n$ 取最值的位置。
先一般性地求 $a_n$。当 $n \geq 2$： $$a_n = S_n - S_{n-1} = \left(-\frac{n^2}{2}+kn\right)-\left(-\frac{(n-1)^2}{2}+k(n-1)\right) = k - n + \frac{1}{2} = \frac{2k-2n+1}{2}$$
$S_n$ 取最大值的条件：$a_n \geq 0$ 且 $a_{n+1} \leq 0$，即项从正变负的转折点。
$$a_n \geq 0 \iff 2k - 2n + 1 \geq 0 \iff n \leq k + \frac{1}{2}$$
因为 $n$ 是正整数且 $k \in \mathbf{N}_+$，所以 $a_n \geq 0$ 当且仅当 $n \leq k$。
从而 $S_n$ 在 $n = k$ 处取最大值： $$S_k = -\frac{k^2}{2} + k^2 = \frac{k^2}{2} = 8 \implies k = 4$$
此法的优点：从 $a_n$ 的正负性直观理解"为什么 $S_n$ 在 $n=k$ 处最大"——因为前 $k$ 项都是正的（在累加），第 $k+1$ 项起变负（开始拖后腿）。

三、知识点拆解

数列 数列 > 递推关系 > $S_n$ 与 $a_n$ 的关系：$a_n = S_n - S_{n-1}$（$n \geq 2$）
数列 数列 > 等差数列 > $S_n$ 为二次函数时数列为等差数列
数列 数列 > 求和技巧 > 错位相减法（适用于"等差 $\times$ 等比"型通项）
函数 函数 > 二次函数 > 配方法求最值 / 顶点坐标
数列 数列 > 等比数列 > 前 $n$ 项和公式 $\frac{a_1(1-q^n)}{1-q}$
数列 数列 > 数列最值 > 通过 $a_n$ 正负性判断 $S_n$ 取最值的位置

四、思维方式

【将 $S_n$ 视为 $n$ 的二次函数，配方求最值】
$S_n$ 是关于整数变量 $n$ 的二次表达式，借用连续函数的最值方法
函数与方程
【用 $a_n = S_n - S_{n-1}$ 将前 $n$ 项和转化为通项】
$S_n$ 与 $a_n$ 的差分关系是数列最基本的转化工具
化归转化
【先代入 $a_n$ 化简 $\frac{9-2a_n}{2^n}$ 再求和】
不急于套公式，先化简通项可能大幅降低计算量
整体思想
【识别 $\frac{n}{2^{n-1}}$ 为"一次 $\times$ 等比"结构，使用错位相减法】
$n$ 是等差因子，$\frac{1}{2^{n-1}}$ 是等比因子，乘以公比再作差消去等差部分
递推思想
【用 $a_n$ 正负性判断 $S_n$ 最大值位置】
$S_n$ 递增 $\iff$ $a_n > 0$，当项变负时 $S_n$ 开始递减，直观建立"加正减负"模型
数形结合

五、易错点与陷阱

求 $a_n$ 时忘记单独验证 $n = 1$ 的情况
必须检查 $a_1 = S_1$ 是否与通项公式一致，不一致则需分段表示
$a_n = S_n - S_{n-1}$ 仅对 $n \geq 2$ 成立。本题恰好一致（$a_1 = \frac{7}{2}$ 符合公式），但这不是默认的。
错位相减法中，作差后忽略最后一项 $\frac{n}{2^n}$
$T_n - \frac{1}{2}T_n$ 的结果 = 等比数列部分 $-$ 尾项 $\frac{n}{2^n}$，两部分都不能漏
错位相减的核心：中间项两两相消变为等比数列，但首项和末项不配对，末项需单独处理。
$S_n$ 最值直接令导数为零，忘记 $n$ 必须取正整数
$n$ 是正整数，需在对称轴附近的整数点取最值；本题 $k \in \mathbf{N}_+$ 保证对称轴恰好是整数
若 $k$ 不是正整数（如 $k = 3.5$），最大值在 $n = 3$ 或 $n = 4$ 处取得，需要比较两个值。
化简 $9 - 2a_n$ 时符号出错，得到 $\frac{9 - 2n}{2^n}$ 等错误结果
$9 - 2 \cdot \frac{9-2n}{2} = 9 - (9-2n) = 2n$，注意整体代入后分配律的符号
$a_n = \frac{9-2n}{2}$，代入时 $2a_n = 9-2n$，$9 - 2a_n = 9 - 9 + 2n = 2n$。分步写清楚不容易错。

六、变式题方向

变式一：$k$ 不是正整数
若去掉 $k \in \mathbf{N}_+$ 的条件，改为 $k > 0$，则 $S_n$ 的最大值取在 $n = \lfloor k + \frac{1}{2} \rfloor$ 处， 需要讨论 $k$ 的范围（如 $3 < k < 4$ 时，比较 $S_3$ 和 $S_4$）。这是一道更复杂的分类讨论题。
变式二：改变求和结构
将第 (II) 问改为求 $\sum_{i=1}^n \frac{a_n}{2^n}$ 或 $\sum_{i=1}^n n \cdot 3^n$， 本质不变（仍是错位相减法），但公比不同，考察计算能力。若改为 $\sum \frac{a_n}{n}$，则不再适用错位相减，需要裂项或其他方法。
变式三：$S_n$ 最值条件换为不等式
例如"$S_n \leq 8$ 对所有 $n$ 成立，求 $k$ 的范围"。此时 $\frac{k^2}{2} \leq 8$ 给出 $0 < k \leq 4$（$k \in \mathbf{N}_+$）， 即 $k \in \{1, 2, 3, 4\}$。这类变式把等式条件放宽为不等式，增加了讨论维度。
