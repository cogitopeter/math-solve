# math-solve — 高中数学7维题目分析器

一个 Claude Code 自定义 Skill（Slash Command），专为 **人教A版高中数学（浙江）** 设计。

拍一张题目照片，或直接打字输入题目，自动生成一份包含 **7个分析维度** 的精美 HTML 页面 + 一个持续更新的学习诊断仪表盘。

---

## 它能做什么？

**输入**：一道数学题（图片或文字）

**输出**：

1. **单题分析页**（白色主题，适合打印）—— 7个维度逐层拆解
2. **学习诊断索引**（深色仪表盘）—— 雷达图、热力图、薄弱环节诊断

每做一道题，索引自动更新。做得越多，诊断越准。

---

## 7个分析维度

### 一、解题思路 🧭

> **这道题是怎么"想到"的？**

不是标准答案，而是做题前的真实思考过程——以时间轴形式展示：

- **审题**：提取了哪些关键条件，第一反应是什么
- **尝试**：最初想用什么方法，走到哪一步发现走不通
- **突破**：怎么想到了正确的方法，关键洞察是什么
- **路径选择**：如果有多条路，为什么选了这条

语气像一个有经验的同学在旁边说"我是这么想的"。目标不是给标准路径，而是激发思维模式。

> 时间轴样式，关键洞察节点用实心圆点高亮（如下图）

![解题思路示例](docs/screenshot-thought.png)

---

### 二、解题过程

至少提供 **2种解法**，每种完整步骤：

- 数学公式用 LaTeX 渲染（KaTeX），排版精美
- 标注解法特点：通法 / 巧解 / 计算量对比
- 关键步骤带框高亮

![解题过程示例](docs/screenshot-solution.png)

---

### 三、知识点拆解

格式：`大模块 > 中模块 > 具体知识点`

覆盖人教A版12个主模块：集合与逻辑、函数、三角函数、数列、不等式、向量、立体几何、解析几何、概率统计、导数、复数、计数原理

每道题涉及的全部知识点——包括隐含的——都会被列出，并标注对应章节。

---

### 四、思维方式

逐步标注每个关键节点用了什么数学思维：

```
【设椭圆标准方程】 → 化归转化：将问题纳入标准框架
【恒成立 → 求最小值】 → 逆向思维：翻转问题方向
```

常见思维：数形结合、分类讨论、化归转化、函数与方程、整体思想、构造法、反证法、特殊值法、逆向思维、设而不求……

![思维方式示例](docs/screenshot-thinking.png)

---

### 五、易错点与陷阱

2-4个常见错误，每个都有：

- ❌ 错误做法
- ✅ 正确做法
- 原因分析

---

### 六、变式题方向

2-3个变式，说明：改什么条件 → 变成什么题 → 核心区别

帮你举一反三，同一个知识点换个壳就不会了的问题在这里解决。

---

### 七、难度定位

- 等级：基础(1) / 中档(2) / 中高档(3) / 压轴(4)
- 高考/模考定位（例如"高考解答题第17题水平"）
- 建议用时
- 适合阶段

## 学习诊断仪表盘

每分析一道题，索引页自动更新。仪表盘包含三个视图：

### 题目列表

所有题目卡片式排列，展开可看知识点、思维方式。每张卡片带7个维度的锚点跳转按钮，一键直达。

![题目列表](docs/screenshot-index-list.png)

### 知识模块

按12个数学模块分组，每个模块下列出相关题目、难度标签。快速定位某个模块做过哪些题。

### 学习诊断

- **知识模块雷达图**：形状越饱满覆盖越均衡，一眼看出哪些模块没练到
- **难度分布柱状图**：基础/中档/中高档/压轴各做了多少
- **思维方式词云**：高频思维 vs 低频思维，找到思维盲区
- **模块掌握度排名**：未涉及 → 需加强 → 初步掌握 → 较好，分级诊断
- **做题热力图**：每日做题量日历视图，追踪学习节奏

![学习诊断](docs/screenshot-index-diag.png)

## 安装

1. 将 `gzshuxue.md` 复制到 `~/.claude/commands/` 目录
2. 在 Claude Code 中输入 `/gzshuxue` 并附上题目图片即可

首次使用时会自动在 `~/Desktop/gzshuxue/` 创建工作目录和索引文件。

## 使用方式

```
/gzshuxue [附上题目照片]
```

或发送题目文字后输入 `/gzshuxue`。

触发词：`做这道题`、`分析这道题`、`这题怎么做`

## 技术细节

- 数学公式：KaTeX 0.16.9 CDN 渲染
- 题目页面：纯静态 HTML，白色主题，`@media print` 适配打印
- 索引页面：纯静态 HTML + 内联 JavaScript，深色主题，SVG 可视化
- 无需后端、无需数据库、无需网络（除 KaTeX CDN）
- 所有数据存在本地 HTML 文件中，可离线浏览
