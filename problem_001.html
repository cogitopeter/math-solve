# 001
2026-05-27 来源：解析几何100题精选 中高档

一、解题思路

审题 — 信息量大，需要找主线
题面给了一堆东西：双曲线、右焦点 $F$、左准线上的点 $M$、向量条件 $\vec{MP}=\vec{OF}$、$P$ 在双曲线上、$|PF|=\lambda|OF|$。信息密度很高，不能随便设坐标就开始算。先想清楚：最终要求的是 $e$ 和 $\lambda$ 的关系，所以一切都要往 $e = c/a$ 上靠。
第一想法 — 坐标硬算？
最直接的思路：$M$ 在左准线上，$x_M = -a^2/c$（$y_M$ 未知）；向量条件给出 $P = (x_M + c,\; y_M) = (b^2/c,\; y_M)$。$x_P$ 算出来了，但 $y_M$ 怎么办？它必须让 $P$ 满足双曲线方程，代入后会不会很复杂？我没有马上走这条路，而是想：有没有不需要 $y_M$ 的方法？
关键洞察 — 向量相等 = 平行四边形
$\vec{MP} = \vec{OF}$ 本质上是说"从 $M$ 到 $P$ 的位移"和"从 $O$ 到 $F$ 的位移"完全一样。两个向量相等意味着平行且等长，所以 $OFPM$ 构成平行四边形，直接得到 $|PM| = |OF| = c$，完全绕过了 $y_M$。这一步是全题关键：看到向量相等，先想几何意义，别急着拆坐标。
选择路径 — 怎么引入离心率
现在有 $|PM| = c$ 和 $|PF| = \lambda c$，需要把 $e$ 拉进来。想到两条路：
路径A：焦准性质 $|PF|/d = e$（$d$ 是到对应准线的距离），$M$ 就在准线上，距离好算。
路径B：焦半径公式 $|PF| = ex_P - a$，代入 $x_P = b^2/c$。
路径B更短，但路径A几何画面更完整。考试中优先选短路。
第(II)问 — 向量比例的处理套路
$\vec{DA} = (\sqrt{3}-2)\vec{DB}$ 这种条件，初学者容易直接写坐标然后硬算。更聪明的做法：分量拆开。$D(0, a)$ 在 $y$ 轴上，$x$ 分量直接给出 $x_1 = (\sqrt{3}-2)x_2$。配合韦达定理的 $x_1 + x_2$ 和 $x_1 x_2$，两式相除消掉 $x_2$，只剩 $a^2$。这是解析几何大题的常见链条：向量 → 坐标分量 → 韦达 → 消元。

二、解题过程

解法一：焦点准线性质法 通法
第 (I) 问：推导 $e$ 与 $\lambda$ 的关系
1. 由 $\vec{MP} = \vec{OF}$，知四边形 $OFPM$ 为平行四边形，故 $|PM| = |OF| = c$。
2. 设右准线 $l: x = \dfrac{a^2}{c}$，$PM$（水平方向）交右准线于点 $N$。
3. $M$ 在左准线 $x = -\dfrac{a^2}{c}$ 上，$N$ 在右准线上，两准线之间的距离为 $$|MN| = \frac{a^2}{c} - \left(-\frac{a^2}{c}\right) = \frac{2a^2}{c}$$
4. 因此 $P$ 到右准线的距离 $$|PN| = |PM| - |MN| = c - \frac{2a^2}{c} = \frac{c^2 - 2a^2}{c}$$
5. 由双曲线焦点准线性质（第二定义）：$|PF| = e \cdot |PN|$，又 $|PF| = \lambda|OF| = \lambda c$，所以 $$\lambda c = e \cdot \frac{c^2 - 2a^2}{c}$$
6. 两边乘以 $c$，再代入 $c = ae$： $$\lambda c^2 = e(c^2 - 2a^2) \implies \lambda a^2 e^2 = e \cdot a^2(e^2 - 2)$$ 约去 $a^2 e$，得 $$\boxed{e^2 - \lambda e - 2 = 0}$$
第 (II) 问：$\lambda = 1$ 时求双曲线方程
7. 代入 $\lambda = 1$：$e^2 - e - 2 = 0$，即 $(e-2)(e+1) = 0$，解得 $e = 2$。
8. 由 $e = 2$ 得 $c = 2a$，$b^2 = c^2 - a^2 = 3a^2$。双曲线方程为 $$\frac{x^2}{a^2} - \frac{y^2}{3a^2} = 1$$
9. 设直线 $AB$：$y = -a(x - 1)$，代入双曲线方程： $$\frac{x^2}{a^2} - \frac{a^2(x-1)^2}{3a^2} = 1 \implies 3x^2 - a^2(x-1)^2 = 3a^2$$ 整理得 $$(3 - a^2)x^2 + 2a^2 x - 4a^2 = 0 \quad \cdots (*)$$
10. 判别式 $\Delta = 4a^4 + 16a^2(3 - a^2) = 12a^2(4 - a^2) > 0$，需 $0 < a < 2$，且 $a^2 \neq 3$。
11. 由韦达定理： $$x_1 + x_2 = \frac{2a^2}{a^2 - 3}, \quad x_1 x_2 = \frac{4a^2}{a^2 - 3} \quad \cdots \textcircled{1}$$
12. 直线与 $y$ 轴交点 $D(0, a)$。由 $\vec{DA} = (\sqrt{3} - 2)\vec{DB}$，比较 $x$ 分量得 $$x_1 = (\sqrt{3} - 2)x_2 \quad \cdots \textcircled{2}$$
13. 由 $\textcircled{1}\textcircled{2}$ 联立： $$(\sqrt{3} - 1)x_2 = \frac{2a^2}{a^2 - 3}, \quad (\sqrt{3} - 2)x_2^2 = \frac{4a^2}{a^2 - 3}$$ 两式相除： $$\frac{(\sqrt{3}-2)x_2}{\sqrt{3}-1} = 2 \implies x_2 = \frac{2(\sqrt{3}-1)}{\sqrt{3}-2}$$ 回代第一式，化简得 $\dfrac{(4 - 2\sqrt{3})}{\sqrt{3}-2} = \dfrac{a^2}{a^2 - 3}$，其中 $$\frac{4 - 2\sqrt{3}}{\sqrt{3} - 2} = \frac{-2(\sqrt{3}-2)}{\sqrt{3}-2} = -2$$ 故 $-2 = \dfrac{a^2}{a^2 - 3}$，解得 $a^2 = 2$（满足 $0 < a < 2$ 且 $a^2 \neq 3$）。
14. 因此双曲线方程为 $$\boxed{\frac{x^2}{2} - \frac{y^2}{6} = 1}$$
解法二：焦半径公式法 更简洁
第 (I) 问的另一种推导
1. 由 $\vec{MP} = \vec{OF}$，设 $M(-\frac{a^2}{c},\, y_M)$ 在左准线上，则 $$P = M + \vec{OF} = \left(-\frac{a^2}{c} + c,\; y_M\right) = \left(\frac{b^2}{c},\; y_M\right)$$
2. 直接利用焦半径公式（$P$ 在右支上，到右焦点的距离）： $$|PF| = ex_P - a = e \cdot \frac{b^2}{c} - a = \frac{eb^2 - ac}{c}$$
3. 代入 $b^2 = c^2 - a^2$ 和 $c = ae$： $$|PF| = \frac{e(c^2 - a^2) - ac}{c} = \frac{a^2 e(e^2 - 1) - a^2 e}{ae} = a(e^2 - 2)$$
4. 又 $|PF| = \lambda c = \lambda ae$，所以 $$a(e^2 - 2) = \lambda ae \implies e^2 - \lambda e - 2 = 0$$
与解法一相比，这种方法跳过了准线距离的几何分析，直接通过焦半径公式代入坐标计算，步骤更少。第 (II) 问的后续计算相同。

五、易错点与陷阱

准线公式写成 $x = \pm\dfrac{c}{a}$ 或 $x = \pm\dfrac{a}{e}$（和椭圆搞混）
双曲线和椭圆的准线都是 $x = \pm\dfrac{a^2}{c} = \pm\dfrac{a}{e}$，不要记成 $\dfrac{c}{a}$
记忆口诀：准线在焦点和中心之间（对双曲线和椭圆都成立），所以 $\frac{a^2}{c} < c$
韦达定理中忘记除以二次项系数，直接写 $x_1 + x_2 = 2a^2$
方程 $(3-a^2)x^2 + 2a^2 x - 4a^2 = 0$ 的韦达定理需要除以 $(3 - a^2)$
只有 $x^2 + bx + c = 0$ 才能直接用 $x_1+x_2 = -b$；一般形式要先除以首项系数
漏掉 $\Delta > 0$ 和 $3 - a^2 \neq 0$ 的检验，直接给出 $a^2 = 2$
必须验证 $a^2 = 2$ 满足 $\Delta = 12 \times 2 \times (4-2) = 48 > 0$ 且 $3 - 2 = 1 \neq 0$
不验证可能得到无意义的解。考试中这一步通常占1-2分
$\vec{DA}$ 的方向写反：误认为 $\vec{DA} = D - A$
$\vec{DA} = A - D$（从 $D$ 指向 $A$），所以 $x$ 分量为 $x_1 - 0 = x_1$
向量 $\vec{AB}$ 的定义是"终点减起点"，即 $B - A$。$\vec{DA}$ 的终点是 $A$

六、变式题方向

变式一：改变 $\lambda$ 的值
若 $\lambda = 2$，则 $e^2 - 2e - 2 = 0$，$e = 1 + \sqrt{3}$。此时 $b^2/a^2 = e^2 - 1 = 2 + 2\sqrt{3}$，方程结构完全不同。
核心区别：$\lambda$ 变化直接改变离心率，进而改变双曲线开口大小。训练对 $e$ 与图形形态关系的理解。
变式二：向量条件改为模的比值
若将 $\vec{DA} = (\sqrt{3}-2)\vec{DB}$ 改为 $|\vec{DA}| = (2-\sqrt{3})|\vec{DB}|$（注意取绝对值），则 $A$、$B$ 相对于 $D$ 的位置有两种可能（同侧/异侧），需要分类讨论。
核心区别：向量等式包含方向信息，标量比值丢失方向，增加分类讨论。
变式三：换成椭圆
若将双曲线改为椭圆 $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$，焦准关系变为 $|PF| = e \cdot d$（$e < 1$），则第 (I) 问的推导逻辑不变但 $e$ 的取值范围不同。
核心区别：椭圆 $e < 1$，方程 $e^2 - \lambda e - 2 = 0$ 可能无正数解小于1，需要讨论 $\lambda$ 的范围。
